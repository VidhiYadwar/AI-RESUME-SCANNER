An AI-powered resume screening system that analyzes resumes and ranks candidates based on job description matching.

## Features

- Upload multiple resumes
- ATS score calculation
- Candidate ranking dashboard
- Skill analysis
- AI suggestions for improvement
- Visual charts and analytics

## Tech Stack

Python  
Streamlit  
Pandas  
Plotly  
Natural Language Processing  

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the project

streamlit run app.py

## Project Goal

To help recruiters automatically analyze resumes and shortlist the best candidates using AI.
